## How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/zirui0625/zirui0625.github.io.git
   cd zirui0625.github.io
   ```

2. **Edit the content**:
   - Update `index.html` with your paper information
   - Replace placeholder images in the `images/` folder
   - Customize styling in `css/paper.css`

3. **Add your images**:
   - Place your method pipeline image as `images/placeholder-method.png`
   - Place your result images as `images/placeholder-result.png`
   - Add comparison images as needed

4. **Push to GitHub**:
   ```bash
   git add .
   git commit -m "Update paper homepage"
   git push origin main
   ```

5. **View your page**:
   - The homepage will be automatically published at: `https://zirui0625.github.io/`

## Customization

### Updating Paper Information
Edit the following sections in `index.html`:
- **Title**: Change `<h1 class="title is-2 publication-title">`
- **Authors**: Update the authors section
- **Links**: Add links to your paper, code, and dataset

### Adding Results
- Use `<img-comparison-slider>` for before/after comparisons
- Create new result cards by copying the existing structure

### Styling
- Modify `css/paper.css` to customize colors and layouts
- Bulma CSS framework is available for responsive design
